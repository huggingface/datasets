## من الملفات

تستخدم التكوينات لتحميل ملفات البيانات. ويتم استخدامها عند تحميل الملفات المحلية أو مستودع مجموعة البيانات:

- الملفات المحلية: `load_dataset("parquet", data_dir="path/to/data/dir")`
- مستودع مجموعة البيانات: `load_dataset("allenai/c4")`

يمكنك تمرير وسيطات إلى `load_dataset` لتكوين تحميل البيانات. على سبيل المثال، يمكنك تحديد معلمة `sep` لتعريف [`~datasets.packaged_modules.csv.CsvConfig`] التي تستخدم لتحميل البيانات:

```python
load_dataset("csv", data_dir="path/to/data/dir", sep="\t")
```
