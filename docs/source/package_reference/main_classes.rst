``DatasetDict``
~~~~~~~~~~~~~~~~~~~~~

Dictionary with split names as keys ('train', 'test' for example), and :obj:`nlp.Dataset` objects as values.
It also has dataset transform methods like map or filter, to process all the splits at once.
