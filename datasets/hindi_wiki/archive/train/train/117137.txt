गणित में अनन्त श्रेणियों के अभिसरण की जाँच करने वाली विधियों को अभिसरण परीक्षण कहते हैं। ये विधियाँ सशर्त अभिसरण, निरपेक्ष अभिसरण, अभिसरण के अन्तराल या अपसरण की जाँच करतीं हैं। 
S के लिए निश्चित व्यंजक ज्ञात करना सदैव सरल नहीं है। अत: हम यह जानने के लिए कि कोई विशिष्ट श्रेणी अभिसारी है या नहीं, अभिसारी और अपसारी की परीक्षाविधियों का प्रयोग करते हैं। यदि कोई श्रेणी केवल धनात्मक पदों से बनी है, तो किसी पद के उपरांत {Sn} एक वृद्धिमय अनुक्रम होगा और ऐसे वृद्धिमय अनुक्रम के अभिसरण के लिए आवश्यक और पर्याप्त अनुबंध यह है कि यह परिमित हो, अर्थात्‌ एक ऐसी अचर राशि K का अस्तित्व हो कि n के समस्त मानों के लिए Sn < K हो। धनात्मक पदोंवाली श्रेणी के अभिसरण परीक्षण की विधियाँ निम्नलिखित हैं :
इस विधि में, किसी अनुक्रम   { a n  }  {\displaystyle \left\{a_{n}\right\}} के पदों की अन्य अनुक्रम   { b n  }  {\displaystyle \left\{b_{n}\right\}} के पदों से तुलना की जाती है। यदि,
n के सब मानों के लिये,   0 ≤  a n  ≤  b n   {\displaystyle 0\leq \ a_{n}\leq \ b_{n}}, तथा   ∑ n = 1  ∞  b n   {\displaystyle \sum _{n=1}^{\infty }b_{n}} अभिसारी हो तो   ∑ n = 1  ∞  a n   {\displaystyle \sum _{n=1}^{\infty }a_{n}} भी अभिसारी होगी।
किन्तु, यदि,
n के सब मानों के लिये,   0 ≤  a n  ≤  b n   {\displaystyle 0\leq \ a_{n}\leq \ b_{n}}, तथा   ∑ n = 1  ∞  a n   {\displaystyle \sum _{n=1}^{\infty }a_{n}} अपसारी है, तो   ∑ n = 1  ∞  b n   {\displaystyle \sum _{n=1}^{\infty }b_{n}} भी अपसारी होगी।
